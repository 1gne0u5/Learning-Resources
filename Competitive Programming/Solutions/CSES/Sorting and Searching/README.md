**1. Distinct Number : _Count distinct elements in an array_**

----
**Input**  
The first input line has an integer n: the number of values.

The second line has n integers   
x1, x2, . . . , xn.

**Output**

Print one integer: the number of distinct values.

**Constraints**  
1 ≤ n ≤ 2⋅10^5  
1 ≤ xi ≤ 10^9  

**Example**

_Input:_  
5  
2 3 2 2 3  

_Output:_  
2
